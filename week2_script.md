## [Slide 3: Verification Results] (~90초)

이번 주 핵심은 검증입니다. 10가지 물리 체크를 수행해서 **전부 통과**했습니다.

첫째, 쌍극자 1개일 때 해석해 α·E_inc와 일치 — 상대 오차 1.5 곱하기 10의 마이너스 16제곱으로 기계 정밀도 수준입니다.

둘째, 쌍극자 2개일 때 해석해 p = α 나누기 1 빼기 α G 공식과 일치.

셋째, Green 함수 공식과 호혜성 G(r) = G(-r) 확인.

넷째, 선형 시스템 풀이 잔차 확인 — 기계 정밀도.

다섯째, 중심 대칭 배열의 거울 대칭 확인.

여섯째, 상호작용 행렬의 호혜성 A_ij = A_ji.

일곱째, **배열 크기 수렴성**인데, 여기가 가장 흥미로운 결과입니다.

여덟째, 파장 스케일링 불변성.

아홉째, **소광 전력 양성** — Optical theorem에서 파생된 체크로, 모든 배열 조건에서 P_ext가 양수임을 확인.

열째, 비균일 솔버의 균일 극한이 균일 솔버 결과와 정확히 일치.

## [Slide 4: Key Physical Finding — Test 7] (~40초)

특히 일곱 번째, 배열 크기 수렴성 테스트가 흥미롭습니다.

중심 원자의 위상 편차가 N을 늘려가며 어떻게 수렴하는지 봤는데, 수렴 속도가 1/N의 0.26 제곱으로 관측됐습니다.

이론적으로 2D 자유공간 Green 함수는 원거리에서 1/√r로 감쇠하기 때문에, 1D 격자합의 수렴 속도가 1/√N, 즉 지수 −0.5여야 합니다. 관측값 −0.26은 상당히 느린 수렴이고, 이는 2D 시스템의 격자합이 조건부 수렴한다는 물리적 특성을 확인해주는 결과입니다. 3D였다면 1/r 감쇠로 훨씬 빠르게 수렴했을 겁니다.

즉, 단순한 수치 에러가 아니라 2D 물리의 본질적 특성을 코드가 올바르게 반영하고 있다는 뜻입니다.

## [Slide 5: Baseline & Period Sweep] (~50초)

베이스라인 결과는, N=31 균일 배열, P=0.6 람다에서 평균 위상 편차 약 6도, 최대 약 8도가 나왔습니다. 서브파장 주기에서 커플링이 실제로 측정 가능한 수준이라는 첫 증거입니다.

다음 주 준비로 주기 스윕도 돌려봤는데, 흥미로운 현상을 발견했습니다. 주기가 람다의 정수배, P = 1, 2, 3 근처에서 위상 편차가 급증합니다. P = 1.0 람다에서 평균 편차가 27도까지 올라갑니다.

이건 물리적으로 **격자 공명**, Wood anomaly에 해당하는 현상입니다. 주기가 파장의 정수배가 되면 새로운 회절 차수가 개방되면서 격자합이 발산하는 지점입니다. 의도한 결과는 아니지만 물리적 깊이가 더해진 셈입니다.

## [Slide 6: Difficulty] (~30초)

현재 주요 어려움은 두 가지입니다.

첫째, 격자 공명 때문에 Δφ(P) 곡선이 단조롭지 않아서, 단순히 "주기 커지면 커플링 작아진다"고 주장할 수 없습니다. 공명 영역과 비공명 영역을 구분해서 분석해야 합니다.

둘째, 방금 언급한 2D 시스템의 느린 수렴 때문에 유한 배열의 edge effect가 3D보다 오래 지속됩니다. 이건 주기 경계 조건이나 Ewald 합법 같은 고급 기법이 필요할 수 있는 부분입니다.
